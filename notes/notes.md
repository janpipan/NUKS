# MongoDB

Connecting to mongoDB

`$ mongosh`

Display all databases

`$ show databases`

Change database

`$ use <dbName>`

Display all data

`$ show db.<collectionName>.find().pretty()`

# PostgreSQL

Connecting to postgreSQL

`$ psql -h localhost -U admin --password -p 5432 polls`

# React

Start react project

`$ npx create-react-app <appName>`
